Introduction
Ce manuel pratique a pour but d’expliquer le fonctionnement du site et de permettre à l’équipe de bénévoles d’Idées Madagascar de le maitriser au mieux.
Attention certaines parties expliquent comment modifier des parties du site, il ne faut modifier QUE les endroits indiqués comme illustrés dans les exemples !


Technologies et liens utiles
Le site web a été développé en HTML5 à travers le framework Bootstrap pour la gestion du css c’est-à-dire la gestion de l’apparence des divers éléments composant la page internet.
Bootstrap vous permet de gérer votre page comme une grille où vous organisez les éléments comme des blocs. Pour plus de détails voici le lien vers le site expliquant en détails tout son fonctionnement :Bootstrap
Si vous rencontrez des problèmes avec le HTML ou le css voici un site de tutorat : W3schools ou encore OpenClasseRooms

Quels sont les acteurs de ce site ?
Les visiteurs (internes ou externes à l’école)… et vous bien sûr !!
Les visiteurs peuvent « surfer » sur le site et vous envoyer un mail pour toute question.
De plus les membres de l’association inscris sur le site peuvent poster des articles qui apparaitront sur la page « news » et qui vous permettront de tenir les visiteurs tels que les sponsors, informés.

Droits d’accès
Tous les membres n’ont pas les mêmes droits d’accès. En effet il existe les rôles de rédacteur et d’administrateur.
Les rédacteurs peuvent écrire des articles qui passent par la validation de l’administrateur avant publication sur le site. L’administrateur peut modifier, ajouter ou supprimer des rédacteurs et des inscrits.

Modification des membres de l’association
A la page Philosophie vous trouverez l’ensemble des membres de l’association avec leur nom et portrait.
Dû à la succession des mandats vous pourrez avoir les modifiez suivants la procédure suivante.
Dans le dossier site allez ouvrir le fichier philo.html
Prenons l’exemple de modification de président. Descendez à la ligne qui suit :
<div class="col-md-6">
<img class="portrait-staff col-md-6" src="../images/julie.jpg" alt="Julie" title="Julie Labeyrie">
<h6>Julie Labeurie
Présidente</h6>
</div>
Modifiez aux emplacements en rouge :
<div class="col-md-6">
<img class="portrait-staff col-md-6" src="../images/imageNouveauPres.jpg" alt="PrénomNouveauPresident" title="Prénom Nom ">
<h6>Prénom Nom
Président</h6>
</div>
 De même pour tous les autres membres à modifier. Ne modifiez que les parties en rouge !

Pour remplacer un partenaire par un autre

Exemple : remplacer la FNAC Montparnasse par IONIS.
Vous enregistrez le logo de IONIS dans le dossier images sous le nom nomdelimageIONIS.jpg.
Puis remplacer /images/logofnac.jpg par /images/nomdelimageIONIS.jpg
Puis remplacer FNAC Montparnasse par IONIS group.
Et voilà !

Pour supprimer un partenaire

Toujours à partir du code précédant on suppose maintenant que l’on souhaite retirer Benjamin Opticien. Sélectionner la partie suivante et supprimer la :

<div class="col-sm-4">
<a href="http://www.benjaminopticien.com/"><img src="/images/benjamin.jpg" class="img-rounded"/></a>
</div>
 De même pour la partie suivante :
<div class="col-sm-4 col-sm-offset-1">
<p class="hidden-xs text-center">Benjamin Opticien</p>
</div>
 Les partenaires situés sur la même ligne se décaleront vers la gauche.
